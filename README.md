# User-validation-program

This is a simple Python program that validates user input for a username. It ensures that the username meets specific rules before allowing the user to proceed.

Features

1. Username cannot be longer than 12 characters.
2. Username cannot contain spaces.
3. Username cannot contain numbers or special characters (only letters are allowed).
4. Provides clear feedback to the user if the input is invalid.

Example 

Enter a username: John123
Your username can't contain numbers or special characters.

Enter a username: John Doe
Your username can't contain spaces.

Enter a username: John
Welcome, John!
